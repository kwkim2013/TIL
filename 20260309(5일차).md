### 📚 상세 학습 내용

#### ✔ 1. C++ 기초 문법

* C++ 프로그램의 기본 구조 이해
* 표준 입출력 사용 (`iostream`)

```cpp id="k2m8qp"
#include <iostream>
using namespace std;

int main() {
    cout << "Hello C++" << endl;
    return 0;
}
```

* 변수 및 자료형

  * 정수형 (`int`)
  * 실수형 (`float`, `double`)
* 조건문 및 반복문 기초 개념 이해

👉 C++을 통해 **빠른 실행 속도를 가진 언어의 특징**과
👉 메모리 기반 프로그래밍의 기초 개념을 이해

#### ✔ 2. Linux 기본 명령어

* CLI(Command Line Interface) 환경에서의 작업 수행

```bash id="p1d8ka"
ls        # 파일 및 디렉토리 목록 확인
cd        # 디렉토리 이동
mkdir     # 디렉토리 생성
```

* 파일 및 폴더 구조 이해
* 터미널 기반 작업 방식 경험

👉 GUI가 아닌 **명령어 기반 환경에서의 개발 방식** 이해
👉 개발자에게 필수적인 서버 환경 기초 학습

---

#### ✔ 3. OpenCV 개요

* OpenCV(Open Source Computer Vision Library)의 역할 이해
* 이미지 및 영상 처리 개념 학습

```python id="m4z7yc"
import cv2

img = cv2.imread("image.jpg")
cv2.imshow("Image", img)
cv2.waitKey(0)
```

* 이미지 데이터를 다루는 기본 흐름 이해

  * 이미지 읽기
  * 화면 출력
  * 사용자 입력 대기

👉 컴퓨터가 이미지를 처리하는 방식에 대한 기초 이해
👉 영상 처리 및 AI 분야로의 확장 가능성 확인

* `main` 메서드의 역할 이해
* 프로그램 실행 흐름 파악


* 변수 선언 및 초기화 방법 학습
* 기본 자료형 이해

  * 정수형 (`int`)
  * 실수형 (`double`)
  * 문자형 (`char`)
  * 논리형 (`boolean`)

```java
int age = 25;
double height = 175.5;
char grade = 'A';
boolean isStudent = true;
```

#### ✔ 4. 기본 입출력

* `System.out.println()`을 활용한 출력
* 사용자 입력 개념 이해 (Scanner 기초)

```java
import java.util.Scanner;

Scanner sc = new Scanner(System.in);
int num = sc.nextInt();
System.out.println(num);
```

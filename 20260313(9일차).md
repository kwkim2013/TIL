#### ✔ 2. 함수 정의 및 호출

```python id="k3x9pw"
def greet():
    print("Hello Python")

greet()
```

* `def` 키워드를 사용하여 함수 정의
* 함수 호출을 통해 실행

👉 코드의 흐름을 기능 단위로 나누는 방식 이해

---

#### ✔ 3. 매개변수(Parameter)와 반환값(Return)

```python id="p7v2mx"
def add(a, b):
    return a + b

result = add(3, 5)
print(result)
```

* 입력값(매개변수)을 받아 처리
* 결과값을 반환하는 구조 이해

👉 함수가 단순 실행이 아닌 “데이터 처리 역할” 수행

---

#### ✔ 4. 함수 활용

* 반복되는 로직을 함수로 분리

```python id="z2n6kd"
def print_numbers():
    for i in range(5):
        print(i)

print_numbers()
```

👉 코드의 가독성과 유지보수성 향상

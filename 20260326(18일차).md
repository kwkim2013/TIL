#### 3. 리터럴과 자료형

```

b = 3.14
c = "hello"
d = True
e = None
```

- 리터럴은 변수에 저장되는 값 자체
- 주요 자료형: int, float, string, boolean, None

#### 4. 주요 자료구조

```
# list
a = [1, 2, 3]

# tuple
t = (1, 2, 3)

# set
s = {1, 2, 2, 3}

# dictionary
d = {"name": "kim", "age": 20}
```

- list: 변경 가능, 순서 있음
- tuple: 변경 불가
- set: 중복 제거, 순서 없음
- dictionary: key-value 구조

#### 5. 입출력 (Input / Output)

```
x = 5; y = 10
print("x={}, y={}".format(x,y))

num = input("Enter a number: ")
print(num)
print(type(num))
```

- print()는 출력 함수
- input()은 사용자 입력 (문자열로 반환)
- format()을 통해 문자열 포맷팅 가능

#### 6. 연산자

```
print(3 > 2)   # True
print(5 <= 5)  # True

x = True
y = False

print(x and y)
print(x or y)
print(not x)

```
- 산술, 비교, 논리 연산자 사용 가능
- >=, <=는 "같음 포함"

#### 7. 조건문(if)


#### 8. 반복문 (for, while)

```
for i in range(5):
    print(i)
```

```
i = 0
while i < 3:
    i += 1
```

- for: 정해진 반복
- while: 조건 기반 반복


#### 9. break / continue / pass

```
for i in range(5):
    if i == 3:
        break
```
- break: 반복 종료
- continue: 다음 반복으로 이동
- pass: 아무 작업도 하지 않음

#### 10. 함수 (Function)

#### 11. 매개변수와 인자

```
def greet(name, msg="hello"):
    print(msg, name)

greet("kim")
greet("kim", "hi")
```

- 매개변수: 함수 정의 시 사용하는 변수
- 기본값(default argument) 사용 가능
- 값 전달 방식 다양 (위치, 키워드)

#### 12. 형변환

```

float(3)
```

#### 13. 문자열 포맷팅

```
name = "kim"
age = 20

print("이름: %s, 나이: %d" % (name, age))
print(f"이름: {name}, 나이: {age}")
```

- %s, %d, %f 사용 가능
- f-string 방식이 더 많이 사용됨

#### 14. 함수(function)
 - 특정 작업을 수행하기 위한 명령문(statement)의 그룹
 - 프로그램을 더 작고, 모듈 청크(chunk)로 나눌 수 있음
 - 코드의 중복성 낮추고, 재사용성을 높을 수 있음
 - 선언 양식

```
 def function_name(parameters):
 """ docsting """
    statement(s)
```

- 사용하기 이전에 선언되어 있어야함
- return 명령문 사용 가능함
```
 return [expression_list]
```
- 변수의 적용 범위는 함수 내부와 외부가 별개임
- 사용자가 정의한 함수와 파이썬 내장함수가 존재함

#### 15. function arguments(인자)

 * 선언시에, 인자의 기본값(default value)을 지정해줌
 * Default 이후에 오는 모든 인자는 기본값이 존재해야함

```
def greet(name, msg):
    "" this"""
    print("hello", name + ',' +msg)

greet("monica", "Good morning!")
```

* 반복문 (for, while)

---

### 📚 상세 학습 내용

#### ✔ 1. 리스트(List) 자료구조

* 여러 개의 데이터를 하나의 변수에 저장하는 방법 학습
* 인덱스를 활용한 데이터 접근

```python id="n3k9s1"
numbers = [1, 2, 3, 4, 5]
print(numbers[0])
```

* 리스트 요소 추가 및 수정

```python id="r4x8za"
numbers.append(6)
numbers[1] = 10
```

👉 여러 데이터를 효율적으로 관리하는 방법 이해

---

#### ✔ 2. 딕셔너리(Dictionary) 자료구조

* key-value 구조의 데이터 저장 방식 학습

```python id="t7m2qp"
student = {
    "name": "Kim",
    "age": 25
}
print(student["name"])
```

👉 데이터의 의미를 기반으로 저장 및 접근 가능

---

#### ✔ 3. 반복문 (for문)

* 리스트와 함께 반복문 활용

```python id="p8v3zx"
for num in numbers:
    print(num)
```

👉 동일한 작업을 여러 번 자동으로 수행하는 구조 이해

---

#### ✔ 4. 반복문 (while문)

* 조건 기반 반복 실행

```python id="c2w9je"
i = 0
while i < 5:
    print(i)
    i += 1
```

👉 조건에 따라 반복되는 프로그램 흐름 이해

---

#### ✔ 5. 반복문 + 자료구조 활용

* 리스트와 반복문을 결합한 데이터 처리

```python id="f6y1ks"
numbers = [1, 2, 3, 4, 5]
total = 0

for num in numbers:
    total += num

print(total)
```

👉 실제 프로그램에서 데이터 처리 흐름을 구현하는 기초 능력 습득

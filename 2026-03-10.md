# 컴퓨터
- 정보 처리 시스템
  정보 처리 시스템(Information Processing System)은 조직이나 개인이 유용한 정보를
  얻기 위해 데이터(자료)를 수집, 저장, 가공, 검색, 송·수신하는 기기와 소프트웨어의 조직화된 체계입니다.
  이는 하드웨어, 소프트웨어, 사람, 절차, 데이터를 포함하여 의사결정을 지원하는 유기적인 집합체로 정의합니다.

# 추상화
추상화(Abstraction)는 복잡한 시스템에서 불필요한 세부 사항은 숨기고 핵심적인 개념과
기능만을 모델링하여 코드를 단순화하는 객체 지향 프로그래밍의 핵심 원칙

# 일반화 , 상세화
상세화(Specialization)와 일반화(Generalization)는 객체 지향 프로그래밍(OOP) 및 소프트웨어 모델링에서 
시스템의 복잡성을 관리하고 코드의 재사용성을 높이는 핵심 개념입니다. 이 두 과정은 주로 상속(Inheritance) 
관계를 통해 구현되며, 서로 반대 방향으로 작용합니다.

# CPU(중앙처리장치)
컴퓨터의 '두뇌'로, 소프트웨어 명령어를 해석·연산하고 데이터를 처리하여 모든 하드웨어(프로그램 흐름)을 제어하는 핵심 부품입니다.

# CPU 버스(Bus)
CPU 버스(Bus)는 CPU와 메모리, 입출력 장치 등 컴퓨터의 주요 구성 요소 간에 데이터, 주소, 제어 신호를 주고받는 내부 전송로입니다. 
데이터 이동(데이터 버스), 목적지 지정(주소 버스), 동작 지시(제어 버스)를 통해 병목 현상을 줄이고 고속 데이터 전송을 가능하게 합니다.

# 비트
비트(Bit, Binary Digit)는 컴퓨터가 처리하는 가장 최소의 데이터 단위로, 0 또는 1이라는 두 가지 가능한 상태 중 하나를 나타냅니다. 
1947년 존 와일더 투키가 'binary digits'의 축약어로 고안했으며, 비트가 8개 모이면 1바이트(Byte)가 되어 하나의 문자를 표현할 수 있습니다.

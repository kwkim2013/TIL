* OpenCV 라이브러리를 활용한 이미지 입출력 실습

```python id="m8w4za"
import cv2

img = cv2.imread("image.jpg")
cv2.imshow("Image", img)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

* 이미지 처리 기본 흐름

  * 이미지 불러오기 (`imread`)
  * 이미지 출력 (`imshow`)
  * 키 입력 대기 (`waitKey`)

👉 컴퓨터가 이미지를 데이터로 처리하는 방식 이해

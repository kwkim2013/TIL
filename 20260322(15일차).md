#### ✔ 1. C++ 반복문 및 로직 처리

```cpp id="c8z4qp"
#include <iostream>
using namespace std;

int main() {
    int sum = 0;
    for (int i = 1; i <= 10; i++) {
        sum += i;
    }
    cout << sum << endl;
    return 0;
}
```

👉 반복문을 활용한 데이터 처리 방식 이해
👉 C++의 구조적인 코드 작성 능력 향상

#### ✔ 2. OpenCV 이미지 처리

```python id="t6v2ka"
import cv2

img = cv2.imread("image.jpg")

if img is None:
    print("이미지를 불러올 수 없습니다.")
    exit()

cv2.imshow("Image", img)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

👉 이미지 입출력 및 예외 처리 방식 이해

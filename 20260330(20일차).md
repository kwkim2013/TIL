#### 1. CSS 기본 개념 (Cascading)

- CSS는 **Cascading Style Sheet**의 약자로, 스타일이 위에서 아래로 흐르듯 적용된다.
- 여러 스타일이 충돌할 경우 **우선순위에 따라 적용**된다.

```
p {
  color: blue;
}

p {
  color: red; /* 나중에 작성된 스타일이 적용됨 */
}

```
#### 2. HTML 요소 (div / span)
- `div` : 블록요소 (한줄 전체 차지)
- `span` : 인라인요소 (내용만큼만 차지) 

```
<div>블록 요소</div>
<span>인라인 요소</span>
```

#### 3. CSS 선택자

---
- 전체 선택자

```
* {
  margin: 0;
  padding: 0;
}
```
- 클래스 선택자
```
.box {
  border: 1px solid black;
}
```
- 선택자 묶기
```
h1, p {
  color: blue;
}
```
---
#### 4. CSS 우선순위
- 스타일 적용 순서
 1. `!important`
 2. 인라인스타일
 3. id 선택자(#)
 4. 클래스 선택자(.)
 5. 태그선택자

 ---

 #### 5. 색상 표현 방식(RGB & HEX)
 - 1byte = 8bit -> 256가지 표현 (0~255)
 * RGB 방식
 ```
color: rgb(255, 255, 0);
 ```
- HEX 방식
```
color: #ffff00;
```

- 색상 예시
* 흰색 → #ffffff / #fff
* 검정 → #000000 / #000
* 노랑 → #ffff00 / #ff0
* 파랑 → #0000ff / #00f


 #### 6. text-indent
 - 문단 첫 줄 들여쓰기
 ```
p {
  text-indent: 20px;
}
 ```

#### 7. list-style-type
- 리스트 스타일 변경
```
ul {
  list-style-type: square;
}

ol {
  list-style-type: upper-roman;
}
```

```

		}

			font-size: 1.5em; /* 헤더 크기 */
			font-weight: bold;
			text-align: center;

		}

		h2, p {
			padding: 0px 10px; /* 테두리와 제목 텍스트 사이의 패딩 여백(위아래/좌우) */
			text-align: center;
		}

			font-weight: bold;
		}

</head>

<body>

		<div id="check">

			2시 이후 주문건은 다음날 발송합니다(주말 제외)</p>
			<hr>
			<h2>교환 및 환불</h2>
			<p>불만족시 <span class="accent" style="color: #0000ff">100% 환불</span>해 드립니다<br>
			고객센터로 전화주세요</p>
			<hr>
			<h2>고객센터 </h2>
			<p>0000-0000<br>
			<span class="smalltext" style="font-size: 0.7em;">상담시간 : 오전 9시 ~ 오후 6시 (토/일, 공휴일 휴무)</span></p>
		</div>
	</div>
```

# 📘 CSS 정리 (Cascading Style Sheet)

---

### 💻 학습 성과

- CSS의 기본 개념과 캐스캐이딩 구조를 이해하였다.
- div와 span의 차이를 통해 블록 요소와 인라인 요소의 개념을 익혔다.
- 전체 선택자, id 선택자, 클래스 선택자를 활용하여 다양한 방식으로 스타일을 적용할 수 있게 되었다.
- CSS 우선순위를 이해하여 스타일 충돌 시 어떤 스타일이 적용되는지 판단할 수 있게 되었다.
- RGB와 HEX를 이용한 색상 표현 방식을 이해하고 활용할 수 있게 되었다.
- 텍스트 들여쓰기와 리스트 스타일 변경을 통해 기본적인 UI 스타일링을 구현할 수 있게 되었다.

---

### 💡 느낀 점 / 이해
이번 수업을 통해 CSS의 기본 구조와 스타일 적용 방식에 대해 이해할 수 있었다. 특히 캐스캐이딩과 우선순위 개념이 중요하다는 것을 느꼈다. 단순히 스타일을 작성하는 것이 아니라, 어떤 스타일이 최종적으로 적용되는지를 고려해야 한다는 점이 인상적이었다.

또한 div와 span의 차이를 통해 웹 페이지의 구조를 어떻게 나누는지 이해할 수 있었고, 선택자를 활용하여 특정 요소에만 스타일을 적용하는 방법을 익힐 수 있었다.

색상 표현 방식(RGB, HEX)을 배우면서 웹에서 색상이 어떻게 표현되는지 이해할 수 있었고, 이를 통해 더 다양한 디자인을 구현할 수 있을 것이라고 느꼈다.

전체적으로 이번 학습을 통해 웹 페이지를 꾸미는 기본적인 방법을 익혔으며, 앞으로는 단순한 스타일 적용을 넘어서 더 구조적이고 효율적인 CSS 작성 방법을 고민해봐야겠다고 생각했다.

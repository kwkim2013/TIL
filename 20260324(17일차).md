#### ✔ 1. Java 키 입력 및 출력 (System.in / System.out)

```java id="j2k8fd"
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("숫자 입력: ");
        int num = sc.nextInt();

        System.out.println("입력값: " + num);
    }
}
```

👉 `System.in`을 통한 입력 개념 이해
👉 `System.out`을 활용한 출력 방식 학습
👉 Scanner를 이용한 사용자 입력 처리

#### ✔ 3. 연산자 (Operator)

* 증감 연산자 (`++`, `--`)
* 대입 연산자 (`=`, `+=`, `-=` 등)
* 비교 연산자 (`==`, `!=`, `>`, `<`)
* 논리 연산자 (`&&`, `||`, `!`)
* 조건(삼항) 연산자 (`조건 ? 값1 : 값2`)

```java id="o9z4sd"
int a = 10;
int b = 20;

int max = (a > b) ? a : b;
System.out.println(max);
```

👉 다양한 연산자를 통한 데이터 처리 방식 이해
👉 조건에 따른 값 선택 구조 학습

#### ✔ 4. 비트 연산 및 시프트 연산

#### ✔ 5. 조건문 (if / switch)

```java id="u7m2ka"
int score = 85;

if (score >= 90) {
    System.out.println("A");
} else if (score >= 80) {
    System.out.println("B");
} else {
    System.out.println("C");
}
```

* 단순 if문
* if-else문
* 다중 if-else문

👉 조건에 따른 프로그램 흐름 제어

#### ✔ 6. switch문 및 제어 흐름

```java id="c4t8bn"
int num = 2;

switch(num) {
    case 1:
        System.out.println("1");
        break;
    case 2:
        System.out.println("2");
        break;
    default:
        System.out.println("기타");
}
```

* switch문 구조 이해
* break를 통한 탈출
* case 값 설정 방식 학습

👉 다양한 조건 처리 방식 비교 이해

* 오버라이딩: 부모 메소드를 자식 클래스에서 재정의

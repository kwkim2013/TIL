### 📚 상세 학습 내용

#### ✔ 1. 배열(Array)

```java
int[] numbers = {1, 2, 3, 4, 5};

for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}
```

👉 여러 데이터를 효율적으로 관리하는 방법 학습
👉 반복문과 배열의 결합 이해

---

#### ✔ 2. 조건문 심화

```java
int score = 85;

if (score >= 90) {
    System.out.println("A");
} else if (score >= 80) {
    System.out.println("B");
} else {
    System.out.println("C");
}
```

👉 다양한 조건 처리 로직 구성 능력 향상

---

#### ✔ 3. 프로그램 작성 실습

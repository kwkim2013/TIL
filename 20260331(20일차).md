### 📚 상세 학습 내용

#### ✔ 1. 조건문 (if / else)

- 조건에 따라 프로그램의 흐름을 제어하는 방법 학습

```java
int score = 90;

if (score >= 90) {
    System.out.println("A");
} else {
    System.out.println("B");
}
```

#### ✔ 2. 리스트(List)
numbers = [1, 2, 3, 4, 5]

for num in numbers:
    print(num)

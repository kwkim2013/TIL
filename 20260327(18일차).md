#### 1. C++ 기본 구조
- c++ 프로그램은 `main()` 함수에서 시작된다.
- `iostream` 라이브러리를 통해 입출력을 수행한다.

```
#include<iosstream>
using namespace std;

int main() {
  cout << "Hello World";
  return 0;
}
```
- #include <iostream> : 입출력 라이브러리
- using namespace std; : std 생략 가능
- main() : 프로그램 시작 함수
- cout : 출력
- endl : 줄바꿈
- cin : 입력받는 명령어 (키보드 입력)
---

#### 2.입력 / 출력

- 출력(`cout`)
```

```

- 입력 (`cin`)
```

```

#### 5. 조건문 (if)
```

  cout << "크다";

  cout << "작다";
}
```
---
#### 6. 반복문(for문, while문)

- for문
```
for (int i = 0; i <5; i++) {
  cout << i << endl;
}
```

- while문
```
int i = 0;
while (i < 5) {
  cout << i << endl;
  i++;
}
```

#### 7. 배열
- 같은 자료형의 데이터를 여러 개 저장

```

#### 8. 함수
- 특정 기능을 수행하는 코드 블록
```

  return a + b;
}

---

#### 9. 포인터
- 변수의 주소를 저장
```
int a = 10;
int* p = &a;

cout << p; // 주소
cout << *p; // 값
```

---
#### 10. 참조자(Reference)
```
int a = 10;
int& ref = a;

ref = 20;
cout << a; // 20
```

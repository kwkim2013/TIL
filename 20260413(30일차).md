---

### 📚 상세 학습 내용

#### ✔ 1. 네비게이션 메뉴 구성 (Navigation Bar)

- 웹 페이지 상단 메뉴 구조 설계
- 링크를 이용한 메뉴 구성

```
<div class="topnav">
  <a href="#">경복궁 소개</a>
  <a href="#">관람안내</a>
  <a href="#">행사일정</a>
</div>

```
---

#### ✔ 2. 검색창(Input) 및 버튼 스타일 적용
검색 입력창과 버튼 UI 구성
CSS를 활용한 버튼 디자인

```
.search-btn {
  float: right;
  padding: 6px 12px;
  background-color: #c6a05e;
  color: white;
  border-radius: 4px;
}
```

#### 3. 검색 아이콘 클릭 기능 구현
input 내부에 돋보기 이미지 추가
클릭 가능한 검색 기능 UI 구성

```
<input type="text" placeholder="궁 이름 검색">
```

👉 사용자 편의성을 고려한 인터페이스 설계 경험

#### ✔ 4. 햄버거 메뉴 버튼 구현
모바일 및 반응형 웹에서 사용하는 메뉴 버튼 개념 이해
메뉴 아이콘을 이용한 UI 구성

```
<button class="menu-btn">☰</button>

```

---

#### 5. Language 선택 메뉴 추가 (한 / 미)
다국어 선택 UI 구성
Header 영역에 Language 옵션 추가

```
<div class="language">
  <a href="#">KOR</a>
  <a href="#">ENG</a>
  <a href="#">JPN</a>
</div>

```
---

다국어(Language) 선택 UI 구현 가능
사용자 중심 UI 설계 경험 확보

### 💡 느낀 점 / 이해

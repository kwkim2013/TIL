### 📚 상세 학습 내용

#### ✔ 1. HTML 주요 태그 심화 학습

* 웹 페이지 구성에 자주 사용되는 태그 학습

```html id="q9x2lp"
<h1>제목</h1>
<p>문단 내용</p>
<a href="https://example.com">링크</a>
<img src="image.jpg" alt="이미지">
```

* 각 태그의 역할 이해

  * `<h1>`: 제목 표시
  * `<p>`: 문단 구성
  * `<a>`: 페이지 이동
  * `<img>`: 이미지 삽입

👉 웹 페이지의 기본 콘텐츠를 구성하는 핵심 요소 학습

---

#### ✔ 2. 속성(Attribute) 활용

* 태그에 추가 정보를 부여하는 속성 개념 이해

```html id="m8v4tr"
<a href="https://google.com" target="_blank">구글 이동</a>
<img src="image.jpg" alt="설명 이미지">
```

* 주요 속성

  * `href`: 링크 경로
  * `src`: 이미지 경로
  * `alt`: 이미지 설명

👉 태그 기능을 확장하는 방법 이해

---

#### ✔ 3. 웹 콘텐츠 구조 설계

* 단순 태그 나열이 아닌 구조적으로 페이지 구성

```html id="y6k3zs"
<header>
  <h1>웹 사이트</h1>
</header>

<section>
  <p>본문 내용</p>
</section>

<footer>
  <p>저작권 정보</p>
</footer>
```

👉 사용자에게 읽기 쉬운 구조 설계 방법 학습
👉 유지보수에 유리한 코드 작성 방식 이해

---

#### ✔ 4. 링크 및 페이지 연결

* 내부/외부 링크 연결 방식 이해

```html id="u1d9wn"
<a href="about.html">소개 페이지</a>
```

👉 여러 페이지를 연결하여 웹사이트 구조 확장 가능

* HTML은 단순한 코드 작성이 아닌 “구조 설계”라는 점을 이해
* 태그를 어떻게 배치하느냐에 따라 가독성과 유지보수성이 달라짐을 체감
* 웹 페이지는 단일 페이지가 아닌 여러 페이지가 연결된 구조라는 것을 이해

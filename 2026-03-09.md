## 📌 클라이언트 / 서버 정리
### 🖥️ 클라이언트
- 서비스를 **요청(Request)** 하는 쪽
- 예: 크롬 브라우저, 스마트폰 앱, PC 프로그램 등

### 💾 서버
- 요청을 받고 **응답(Response)** 해주는 쪽
- 예: 웹 서버, 데이터베이스 서버, 게임 서버 등

## 📚 추가 기능: 네트워크 프로토콜 간단 요약
### 🔹 HTTP
- 웹페이지를 주고받는 프로토콜

### 🔹 FTP
- 파일을 전송할 때 사용하는 프로토콜

### 🔹 TCP
- 연결 기반, 신뢰성 높은 데이터 전송

### 🔹 UDP
- 비연결 기반, 빠른 속도 but 신뢰성 낮음 (영상/게임 등에 사용)

### 🔹 IP
- 인터넷에서 목적지로 패킷을 전달하는 주소 규칙
